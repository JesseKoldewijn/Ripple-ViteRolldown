## Getting Started

### Prerequisites

- Node.js 24.8.0 (managed with Volta)
- Yarn 4.10.2

### Installation

```bash
yarn install
```

### Development

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Preview

```bash
yarn preview
```

### Format Code

```bash
yarn format
```
